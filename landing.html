What it does

Check

Risk-score an agent action before it runs. APPROVE or REJECT in milliseconds.

Predict

Blind-predict outcome before execution. Compare with actual. Evolve.

Audit

Scan execution traces. Find errors, warnings, slow steps.

Gene Map

Error memory. Same error, 1ms lookup, $0 re-diagnosis.

Dashboard

System state. Pending actions. Drift visualization. Live.

Evolve

Observations lifecycle. Promote what works, archive what doesn't.

Why

987+ repos on GitHub are building agent eval frameworks. Patronus just raised $50M for agent simulation. But nobody stops the agent BEFORE it breaks production.

agent-lint does.
